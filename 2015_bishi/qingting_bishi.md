### 1 求复杂度

求下面函数的时间复杂度

``` C
int f(unsigned int n)
{
	if(n <= 0)
		return 0;

	if(n == 1)
		return 1;

	return 5 * f(n - 1) - 7 * f(n - 2);
}
```

解答：

通过上式，可以知道：

当n <= 1时，T(n) = O(1)；

当n > 1 时，T(n) = 5 * T(n - 1) + 7 * T(n - 2) + O(1)；

因此，有

![](递归式图片)

由上图可以知道：

```
T(n) = O(1) + 12 * O(1) + 12 ^ 2 * O(1) + 12 ^ 3 * O(1) + ...
T(n) = 12 ^ n * O(1)
```

因此T(n) = O(12 ^ n)

```
(1) 清零：X = 0;
(2) 变量赋值：X1 = X2;
(3) 嵌套循环：
Loop(X1)
	...
	Loop(X2)
	...
	END
END
```

1 常量不能出现“0”以外的数字。
2 该脚本不支持任何比较运算或if语句。

```
X = 0;
X++;
X++;
Loop(x)
	X = 0;
END
```

(2) 如果已经实现(1)，请实现功能X1 = X1 - X2，若X1 < X2，执行完后X1 = 0；(可利用已经实现的功能)
