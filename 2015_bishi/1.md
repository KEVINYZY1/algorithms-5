## 删除链表中的重复元素(创新工厂2014/9/10)

> 给定一个单链表，删除链表中的重复元素。

``` C++
struct list_node {
	int data;
	list_node *next;
};

void remove_duplicates(list_node *head)
{
	if(head == NULL || head->next == NULL) {
		return;
	}

	bool is_remove = false;

	list_node *pre = head;
	list_node *pnode = pre->next;;

	while(pnode != NULL) {

		if(pre->data == pnode->data) {
			is_remove = true;

			pre->next = pnode->next;
			delete pnode;
			pnode = pre->next;
		}
		else {
			if(is_remove) {
				pre->data = pnode->data;
				pre->next = pnode->next;
				delete pnode;
				pnode = pre->next;
				is_remove = false;
			}
			else {
				pre = pnode;
				pnode = pnode->next;
			}
		}
	}
}
```
